# Timer

This is a simple, graphical count down timer.

## Dependencies

1. Qt, version 6.11
1. CMake
1. VS 2022

## Details

This no longer uses a .pro file and QMake.

Instead, I used and adapted the boiler plate <code>CMakeLists.txt</code> for standard Qt projects.

I also updated some of the code to handle API changes moving to Qt 6
